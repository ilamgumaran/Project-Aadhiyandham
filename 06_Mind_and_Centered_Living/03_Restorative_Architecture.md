# Restorative Architecture: Pattern Language

Designing the environment to support the "Centered Null."

## Foundational Patterns
*   **Light on Two Sides:** Every room must have windows on two sides to eliminate glare and balance the nervous system.
*   **The Alcove:** Small, protected "nooks" for contemplation.
*   **The Window Place:** A dedicated place to sit directly against a window to connect with outdoor rhythms.
*   **Compression:** Using only essential elements to create a high-meaning, minimalist sanctuary.
