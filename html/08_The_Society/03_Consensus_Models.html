1. Collective Agreement

Consensus is a process where a decision is built together until everyone can "live with it." It prevents the fractures caused by majority rule.

The Consensus Flow

PROPOSAL -> CLARIFICATION -> TEMPERATURE CHECK -> MODIFICATION -> CONSENSUS

Temperature Check (Thumbs)

  • Thumb Up: I support this.
  • Thumb Sideways: I have concerns but can live with it.
  • Thumb Down: I BLOCK this (Critical principle violation only).

How-To: Craft a Talking Stick

The physical anchor of the consensus circle, ensuring everyone is heard:

  • Material: 30cm branch of resilient local hardwood (Oak/Cedar).
  • Design: Sand smooth and carve symbols of the Centered Null.
  • Usage: In a meeting, only the holder may speak. It regulates the pace and prevents emotional hijacking of the discussion.

The Facilitator's Role

The facilitator is a servant of the process, ensuring the Talking Stick is respected and every concern is heard and integrated.

🚀 Practical Implementation Guide for Beginners

1. Step-by-Step Action Plan

  • Preparation: Read this module fully before attempting to build or implement. Gather raw materials locally.
  • Execution: Build a small-scale prototype first to test physics and material strength.
  • Review: Test the implementation under stress (wind, water, heavy use) and refine.

2. Troubleshooting & Failure Modes

  • System Failure: When a system breaks, apply the "5 Whys" technique to find the root cause. Fix the root cause, not just the symptom.
  • Burnout: Decadal survival is a marathon. If exhausted, focus on rest and social cohesion.

3. Community & Decadal Flourishing

  • Apprenticeship: No critical skill should be held by only one person. Document local modifications in a physical journal.
